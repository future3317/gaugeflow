TV cannot be interpreted as an optimization-only failure. The qualified H0
split deliberately makes matcher envelopes disjoint, and those envelopes
contain reduced anonymous stoichiometry. Train/validation conditional integer
partition TV is therefore exactly `1.0`; conditional support-size TV is
`0.552677`. The partition-key encoding is injective over all 1,840 supported
partitions.

The H0 split remains valid for OOD novelty and coverage. IID distribution
calibration must use a train-internal reference without adding validation rows
to training. See the successor factorized stoichiometry-first report for the
current implementation and result.
